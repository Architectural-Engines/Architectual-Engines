**Sandbox Wallet Setup for Ledger Testing**:

This folder contains scripts and configurations for testing cryptographic ledger transactions in a controlled environment.

**Purpose**:

-Generate ephemeral wallets for experimentation.

-Simulate transactions without any real funds or live network interactions.

-Enable safe testing of ledger logic, transaction flows, and verification.


**Instructions**:

=Run generate_test_wallets.py to create a set of wallets.


-Assign mock balances using initialize_mock_balances.py.


-Use test_transactions.py to simulate transfers, ledger updates, and validations.


-Inspect logs or outputs to verify behavior.



**Important**: 

**These wallets are sandbox-only. No real assets are used or affected.**
