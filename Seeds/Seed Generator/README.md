***How to basic guide***:

  Seed generator **was/is** used to map intial seeds.  Choice of seed is dependent on use case and exploration.  
    -Once generated, most stable seeds are recommended at earlier stage: tuning parameters.
      note:
        **stable seeds**: resemble a small tight ball, closer to the center of the graph higher stability.
            **supplied**:50 examples of seeds of each movement type linear, Oscillatory, circular, random walk. 

            Higher blue/Less red showing. The more stable
            
        **bifurcation seeds**: these seeds exist as the predicted path and actual path looking as if they are pulling apart or divided. 
            Might resemble wedges, candle flames, triangles, rectangles.
          

    ** Do not build from the seed model without a back up.  this is to return the model to base state and calibrate. **
